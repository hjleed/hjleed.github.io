Project Components & Deliverables

Students are required to submit the following four mandatory deliverables for the final course project. Each component contributes to the overall project grade and must be submitted by the specified deadlines.

  • Project Proposal: A written document outlining the problem statement, project objectives, dataset(s), proposed methodology, and expected outcomes.
  • Project Code: Complete, well-documented Python source code implementing the proposed solution. The code must run correctly and reproduce all reported results.
  • Final Project Report: A structured technical report including an introduction, related studies, methodology, experiments, results, and conclusion. The report must follow academic writing standards.
  • Presentation: A concise oral presentation summarizing the project motivation, approach, experimental results, and key conclusions.
